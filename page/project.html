Projet Tp_Fil_Rouge

Ce projet consistait en développement de site ecommerce qui sera nourrit en HTML, CSS, Javascript et Php.
Pour ce faire j'ai d'abord créer une maquette sous adobe Xd pour recenser les élements dont je voulais faire figurer sur le site. Par la suite, j'ai entamé le développemnt du site, en commençant par le Header constitué du nom site à gauche et des volets de navigation Accueil, Produit et Formulaire pour enregistrement des produits par l'admin du site. Dans le Header on retrouve également une barre de recherche pour filtrer les produits, un espace de connexion et l'ajout de favoris. L'image suivant vous montre un visuel du site.

Projet Supmarket

Supmarket est un logiciel desktop de gestion de magasin que j'ai développé avec l'aide d'un binome pour ce projet. les missions étaient de réaliser l'inventaire de stock, de réaliser une vente, passer une commande fournisseur et de mettre une alerte sur la date de péremption des produits.
Pour ce faire, nous avons modélisé des diagrammes de cas utilisateurs, d'activés pour avoir l'interaction avec le système. Pour les données, nous avons modélisé un modèle conceptuel de données pour connaître l'interaction avec les différentes tables de la base de données.
Le développpement de cette application s'est faite sous Angular 11 pour le côté front end, et Laravel 8 pour gérer le Back end. J'étais en charge de la partie Back end de ce projet mais j'intervenais aussi sur certaines fontionnalités du système partie front end. L'image ci-dessus concerne la page de passation de commande.

Projet Supmarket Mobile

Cette partie du projet supmarket concerne le volet mobile. Pour cette partie il était question de permettre aux autres employés du magasin les rayonnistes, de participer à l'inventaire de stock via leur smartphone en scannant le code barre produit. La deuxième fonctionnalité de l'application mobile, était de créer une vue sur le état de mon stock (Ventes du mois, les bénéfices et pertes).
